# Downloads Folder

This folder contains PDFs and documents that are linked from the "Formulare und Flyer" dropdowns.

## Kieferorthopädie Documents:
- `anamnesebogen-kieferorthopaedie.pdf` - Patient intake form
- `aufklaerungsbogen-kieferorthopaedie.pdf` - Informed consent form  
- `nachsorge-anleitung.pdf` - Aftercare instructions
- `kostenplan-vorlage.pdf` - Cost plan template
- `informationsflyer-kieferorthopaedie.pdf` - Information brochure

## CMD Documents:
- `cmd-fragebogen.pdf` - CMD questionnaire
- `schmerzskala-cmd.pdf` - Pain scale for CMD
- `cmd-behandlungsplan.pdf` - CMD treatment plan
- `cmd-uebungen.pdf` - CMD exercises guide
- `cmd-informationsblatt.pdf` - CMD information sheet

## OSAS Documents:
- `epworth-skala.pdf` - Epworth Sleepiness Scale
- `schlaftagebuch.pdf` - Sleep diary
- `schnarch-fragebogen.pdf` - Snoring questionnaire
- `schienenpflege-anleitung.pdf` - Splint care instructions
- `osas-informationsbroschuere.pdf` - OSAS information brochure

## Instructions:
1. Add your PDF files to this folder with the exact filenames listed above
2. Or modify the links in the HTML files to match your preferred filenames
3. All links open in new tabs/windows (`target="_blank"`)
